Transference

Introduction

Transference is a utility to help you copy & move files (and folders) quickly and precisely. Unlike Finder, which uses multiple windows and dragging, Transference is based around a 2-pane metaphor. In the left pane you specify the files to be transferred, in the right pane you specify the destination, then you trigger the transfer. Simple! No more dropping things on the wrong folder, or having the folder window disappear, or spring open in the wrong place, or any of the myriad other ways Finder can trip you up.

The Browser Window

Transference is based around the concepts of a source-pane (the left pane) and a destination-pane (the right pane) with files flowing from left to right, source to destination. The source and destination are highlighted in the status area at the bottom of the window. These panes can be resized by dragging the divider between them.

Every file and folder in the source pane has a checkbox next to it. Transference will only copy or move files that have been checked. In Finder it can be difficult to select multiple files accurately so that you either have to try and maintain a tricky multiple-selection or do many smaller drag operations. Transference makes this easy, you go down the source list checking off the things you want to transfer and then trigger the transfer at the end.

The destination pane displays only folders that can be the target of files from the source pane. This makes it easier to pick out the right folder.

Navigating the Filesystem

There are several ways to navigate to the folders you want to work with to reflect the different styles and experience of Mac users. Each pane has a field with the current path displayed in it with a button (marked ...) to it's right. You can either type in a path or click the button to get a default MacOS X brower dialog.

When typing paths into the field you can use the F5 or Escape keys to have Transference auto-complete the path for you. Transference will not allow you to enter an invalid path. You can also drop down a list of previous paths for that browser by clicking the drop-down button immediately to the right of the field.

In both browsers you can double-click any folder to drill-down into the contents of that folder. At the bottom of the window Transference indicates where files will be transferred from and to. By default the destination is the folder specified in the destination field however you can quickly specify one of the sub-folders by clicking it. This makes it easy to transfer files between a source and a set of sibling folders.

There are also toolbar buttons (and menu items) for selecting the home, root, parent, and previous folders for the currently selected browser.

Selecting and Transferring

Specifying files to be transferred is done by putting a checkmark next to them. You can check a single file or any combination of files. In either table of files you can multiple-select rows and then use either the toolbar select/deselect buttons or the keyboard shortcuts * to skip, - to deselect, + (or Space) to select, and Enter to toggle. On an external keyboard this allows you to process the entire file list quite rapidly.

In folders with many files it can be hard to select just the ones you want. Transference has a filter field that you can use to narrow the files you want to work with. When you type a string of characters into the filter field Transference will hide any file whose name does not contain that string. Note that Transference will only transfer files that are not currently filtered regardless of whether they have been checked.

Once you have checked the files to be transferred use either the Copy or Move toolbar buttons to do the transfer (alternatively you can use the menu or the keyboard shortcuts Cmd-Opt-C for copy and Cmd-Opt-M for Move). Transference will display a progress dialog so you can see how the transfer is going.

Note that the destination pane shows only folders, not files. You will not be able to see the files that you have copied or moved.

Miscellaneous Features

The toolbar has Follow (Cmd-F) and Switch (Cmd-S) buttons.

Follow makes the destination pane follow the source pane. If you are transferring files to a sub-folder select the source folder in the source pane then follow to make the destination pane match up. You can then select the sub-folder in the destination pane.

Switch makes the source the destination and vice versa. This makes it easy to transfer files between two folders by selecting them and then switching to copy files in either direction.
